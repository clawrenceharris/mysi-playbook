# MySI Playbook

MySI Playbook is a web application for Supplemental Instruction (SI) leaders to create, manage, and run interactive learning sessions. The platform enables:

- **Playbook Management**: Create and organize lesson plans with teaching strategies
- **Session Management**: Schedule and run live SI sessions with video calling capabilities
- **Interactive Activities**: Execute teaching strategies with real-time participant engagement
- **Role-based Access**: Support for SI leaders, students, and coordinators with different permissions

The application focuses on both in-person and virtual learning environments, with features like breakout rooms, participant management, and strategy execution tools.

## Core Entities

- **Playbooks (Playbooks)**: Collections of teaching strategies organized by topic
- **Sessions**: Live or scheduled SI sessions linked to playbooks
- **Strategies**: Individual teaching activities with step-by-step instructions
- **Profiles**: User accounts with role-based permissions (si_leader, student, coordinator)
